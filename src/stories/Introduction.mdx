# AGH UI

AGH UI is a modern React design system built with TypeScript, Vite, Storybook and CSS Variables.

It provides reusable, themeable and accessible UI components for building consistent web applications.

---

## Goals

- Build consistent interfaces faster
- Centralize design decisions
- Improve developer experience
- Support light and dark themes
- Keep components simple, reusable and maintainable

---

## Tech Stack

- React
- TypeScript
- Vite
- Storybook
- CSS Variables
- BEM CSS architecture

---

## Current Components

- Typography
- Icon
- Button
- Badge
- Input
- Textarea
- Checkbox
- Radio
- Select
- Card

---

## Design Principles

### Token-first

Colors, spacing, typography and radius are managed through CSS variables.

### Accessible by default

Components include semantic HTML, focus states, disabled states and error states.

### Themeable

The system supports multiple themes using CSS variables.

### Simple API

Each component exposes a small and predictable API.
