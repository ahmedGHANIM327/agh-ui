# 🚀 AGH UI

A modern React Design System built with Vite, TypeScript, Storybook and CSS Variables.

---

## Why this project exists

AGH UI is not only a component library.  
It is a scalable design system focused on consistency, accessibility, theming and developer experience.

## Core principles

- Design tokens first
- Component consistency
- Accessibility by default
- Themeable architecture
- Framework-friendly React components
- No utility-class dependency
- Clear CSS architecture


## Foundations

- Colors

## Theming

The system supports multiple themes:

- Default
- Green
- Purple
- Dark mode

## Architecture

Each component follows the same structure:

```
components/button/
├── Button.tsx
├── Button.css
└── Button.stories.tsx
```

## Accessibility

Components are designed with:

- semantic HTML
- keyboard support
- focus-visible states
- disabled states
- error states
- labels and descriptions

## Storybook

Storybook is used as the design system documentation platform.

Each component includes:

- default state
- variants
- disabled state
- error state
- controlled examples
- theme examples
